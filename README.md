# learn_java
各个项目对应关系

```
base_frame : 
    基础平台架构，包括  springmvc   mybatis  generator  maven  shiro   ehcache   spring   mysql  pagehelper , dwr等知识
    
-----------------------

blog:
    个人博客搭建  zhangjy520.github.io   包括：hexo等知识
    
-----------------------

data_hub:
    数据推送中心，通过表管理，字段管理，合理安排推送方式。包括：activeMq，jms等知识

-----------------------

gkOpen:
    开放平台，包括应用注册审核，推送等功能。和data_hub公用数据库

-----------------------

gkPlatform:
    云平台，各种功能

-----------------------

make_trigger:
    配置表，生成该表的日志表，触发器。在更改该表的数据的时候，操作日志同步到生成的表中
    
-----------------------

node_pa_chong：
    nodejs 爬取第三方页面

-----------------------

smart_frame:
    从0开始写springMvc，aop，ioc等的实现原理
    
-----------------------

smart_ring:
    智能感知手环
-----------------------

spring_boot:
    springBoot   itext，hcavans  截图网页部分并保存为图片或者pdf

-----------------------

tencentCopy:
    tencent复制版本，docker部署模板 nginx等

----------------------- 

```
